# تشغيل المشروع على XAMPP

## المتطلبات
- XAMPP
- متصفح

## خطوات التشغيل

### 1) موقع المشروع
الأفضل نسخ **محتويات مجلد `dev/`** إلى:

```text
C:\xampp\htdocs\hevan-booking
```

> البنية الصحيحة تعني وجود ملفات مثل `index.php` و `database.sql` مباشرة داخل المجلد. لو نسخت كامل الريبو، افتح غالباً: `http://localhost/hevan-booking/dev/`.

### 2) تشغيل الخدمات
يتطلب التشغيل تفعيل:
- Apache
- MySQL

### 3) إنشاء قاعدة البيانات
تتم العملية عبر:

```text
http://localhost/phpmyadmin
```

ويكون اسم القاعدة:

```text
hevan_booking
```

### 4) استيراد قاعدة البيانات
يتم استيراد الملف التالي داخل القاعدة:

```text
database.sql
```

### 5) فتح المشروع
يكون رابط التشغيل:

```text
http://localhost/hevan-booking
```

## دخول الإداري
- username: `admin`
- password: `admin123`

## ملاحظات عند عدم التشغيل

### خطأ قاعدة البيانات
غالباً يكون السبب:
- MySQL غير شغال
- اسم القاعدة مختلف عن `hevan_booking`
- `database.sql` لم يتم استيراده

### خطأ 404
غالباً يكون السبب:
- المشروع خارج `htdocs`
- اسم المجلد لا يطابق الرابط
- نسخت كامل الريبو ثم فتحت رابط بدون `/dev/`

### تعذر تشغيل Apache أو MySQL
غالباً يوجد تعارض بورت مع برنامج آخر، ويعالج من XAMPP.
