# Hevan Booking - نظام إدارة الحجوزات السياحية

**مشروع طلابي - PHP + MySQL - للمناقشة**

نظام بسيط لإدارة الشركات السياحية، عرض المناطق السياحية، وتسجيل الحجوزات.
مبني على مشروع بحثي أكاديمي (PDF مرفق) ويطبق عملياً المفاهيم النظرية.

---

## 📋 متطلبات التشغيل (XAMPP على ويندوز)

1. **XAMPP** (أي بديل: WAMP, Laragon, MAMP)
2. متصفح ويب

---

## 🚀 طريقة التشغيل

### 1. موقع الملفات
هذا الملف يشرح تشغيل **محتوى مجلد `dev/` كجذر للموقع**.

انسخ محتويات `dev/` إلى:
```
C:\xampp\htdocs\hevan-booking
```

> البنية الصحيحة هنا تقتضي وجود ملفات مثل `index.php` و `config.php` و `database.sql` مباشرة داخل `hevan-booking`. لو نسخت كامل الريبو وليس `dev/` فقط، سيكون الرابط غالباً: `http://localhost/hevan-booking/dev/`.

### 2. تشغيل الخدمات
يتطلب التشغيل تفعيل الخدمتين التاليتين من **XAMPP Control Panel**:
- **Apache**
- **MySQL**

### 3. إنشاء قاعدة البيانات
تتم العملية عبر:
```
http://localhost/phpmyadmin
```

ويكون اسم قاعدة البيانات:
```
hevan_booking
```

### 4. استيراد قاعدة البيانات
بعد إنشاء القاعدة يتم استيراد الملف التالي:
```
database.sql
```

### 5. فتح المشروع
عند استخدام اسم المجلد:
```
hevan-booking
```

يكون رابط التشغيل:
```
http://localhost/hevan-booking
```

## ⚡ تشغيل سريع

ملخص التشغيل:

1. انسخ محتويات `dev/` إلى:
   `C:\xampp\htdocs\hevan-booking`
2. تشغيل **Apache** و **MySQL**
3. استيراد ملف:
   `database.sql`
4. فتح الرابط:
   `http://localhost/hevan-booking`

لو تستخدم Docker من جذر الريبو:

```bash
./refresh-docker-web.sh
```

ثم افتح:

```text
http://127.0.0.1:8080
```

ولو المتصفح ما عرض التحديث، استخدم `Ctrl + Shift + R`.

## 🛠️ ملاحظات عند عدم التشغيل

### خطأ اتصال قاعدة البيانات
يرتبط غالباً بأحد الأسباب التالية:
- خدمة **MySQL** غير مفعلة
- قاعدة `hevan_booking` غير موجودة
- ملف `database.sql` لم يتم استيراده

### خطأ 404 أو Not Found
يرتبط عادةً بأحد الأسباب التالية:
- المشروع ليس داخل `htdocs`
- اسم المجلد لا يطابق الرابط المستخدم
- مثال: عند تسمية المجلد `hevan-booking` يكون الرابط `http://localhost/hevan-booking`

### تعذر تشغيل Apache أو MySQL
السبب المعتاد هو تعارض بورت مع برنامج آخر، وفي هذه الحالة تتم معالجة التعارض أو تغيير البورت من إعدادات XAMPP.

### تعذر فتح صفحة الإداري
بيانات الدخول الافتراضية:
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## 📂 هيكل المشروع

```
hevan-booking/
├── index.php          # الصفحة الرئيسية (عرض المناطق والشركات)
├── book.php           # نموذج حجز جديد
├── login.php          # تسجيل دخول الإداري
├── logout.php         # تسجيل الخروج
├── config.php         # إعدادات الاتصال بقاعدة البيانات
├── auth.php           # المصادقة وإدارة الجلسات
├── database.sql       # هيكل قاعدة البيانات + بيانات تجريبية
├── .htaccess          # إعدادات الحماية
│
├── assets/
│   └── style.css      # التنسيقات
│
├── includes/
│   ├── header.php     # القالب العلوي
│   └── footer.php     # القالب السفلي
│
└── admin/
    ├── _layout.php     # قالب صفحة الإداري
    ├── index.php       # لوحة التحكم
    ├── companies.php   # إدارة الشركات (CRUD)
    ├── places.php      # إدارة المناطق السياحية (CRUD)
    └── bookings.php    # إدارة الحجوزات (تحديث الحالة)
```

---

## 🗄️ هيكل قاعدة البيانات

- **users** - المستخدمون (admin)
- **companies** - الشركات السياحية
- **places** - المناطق السياحية
- **bookings** - الحجوزات (pending/accepted/rejected)

البيانات التجريبية مضمنة: 4 شركات، 5 مناطق، 3 حجوزات.

## 🎯 الوظائف

### للمستخدم العادي (الزائر):

- إدارة الحجوزات (تغيير الحالة: انتظار/قبول/رفض، حذف)
